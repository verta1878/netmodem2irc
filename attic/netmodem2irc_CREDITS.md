# netmodem2irc — credits and source provenance

Honoring the chain that made this revival possible. Attribution is the practice:
we record not just who WROTE the code, but who PRESERVED and PASSED IT ON.

## The source chain
- **Dedrick Allen** — author of the original **NetModem/32** (32-bit FOSSIL Telnet
  server, ~1997-2001; GPLv2). The program this project revives. Original
  FILE_ID.DIZ preserved in history/.
- **mag69** — gave the NetModem source code to Antonio Rico (Reapern66 / verta).
  The revival exists because mag69 preserved and passed on the source. Without that
  hand-off there would be nothing to revive. Credited for making netmodem2irc
  possible.
- **Antonio Rico** (Reapern66 / verta1878) — maintainer of the revival; NetModem/32
  beta tester in the original era; carried the work forward (netmodem2irc).

## Companion / bundled work (also credited)
- **Lukas Gebauer** — Ararat Synapse (modified-BSD), the TCP layer.
- **Maarten Bekers** — ELECOM / EleBBS comms library (freeware; credit + send
  changes back).
- **Vyacheslav Frolov** — com0com / com2tcp / hub4com (GPLv2), virtual-COM
  reference for the fallback path.
- The **Free Pascal** team — FPC, the foundation fpc264irc forks.
- **g00r00** — Mystic, and the deliberate choice to release code that makes the
  IRC/A38 work possible.

## The practice
This file exists because preservation without attribution is theft dressed as
rescue. Everyone who wrote, preserved, or passed on a piece of this is named. If
you carry this forward, keep the chain intact and add your own hand-off to it.
